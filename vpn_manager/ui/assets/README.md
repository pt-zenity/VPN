# Logos de los protocolos

El panel muestra el logo del protocolo activo desde esta carpeta:

- `openvpn.svg`
- `wireguard.svg`

Los ficheros incluidos son **marcadores de posición neutros** (un lettermark con el
color de marca). Para mostrar los **logos oficiales** de OpenVPN y WireGuard, sustituye
estos dos ficheros por los SVG/PNG oficiales (manteniendo el nombre del fichero):

- OpenVPN: marca registrada de OpenVPN Inc. — descarga su recurso oficial.
- WireGuard: "WireGuard" es una marca registrada de Jason A. Donenfeld — usa su logo
  oficial según sus indicaciones de uso.

> Se dejan placeholders a propósito: el logo oficial de cada proyecto es una marca
> registrada y no se incluye en el repositorio. Como administrador del software que
> gestionas, puedes colocarlos aquí para identificarlo.

Si usas PNG, cambia la extensión en `index.html` (busca `/assets/${name}.svg`).
